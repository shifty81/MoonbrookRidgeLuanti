🏗️ ARCHITECTURE
════════════════════════════════════════════════════════════════════════

Layer 1: GAME CONTENT (games/moonbrook_ridge/mods/)
  ├─ mbr_core: World blocks (stone, dirt, water, trees)
  ├─ mbr_items: Food and drinks
  ├─ mbr_tools: Mining and gathering tools
  ├─ mbr_mapgen: World generation
  ├─ mbr_content: NPCs, welcome messages, helpers
  ├─ mbr_farming: Crops, soil, watering, seasonal growth
  ├─ mbr_fishing: Fishing rod, fish species, bait
  ├─ mbr_mining: Ores, gems, cave hazards
  ├─ mbr_foraging: Wild herbs, mushrooms, berries
  ├─ mbr_shop: Economy, currency, merchants
  ├─ mbr_station: Crafting station with recipe discovery
  └─ mbr_upgrades: Tool upgrade system with tiers
  
Layer 2: GAME SYSTEMS (games/moonbrook_ridge/mods/mbr_systems/)
  ├─ mbr_utils: Shared utility functions
  ├─ mbr_time: Seasons and time tracking
  ├─ mbr_survival: Hunger and thirst
  ├─ mbr_weather: Dynamic weather
  ├─ mbr_particles: Visual effects
  ├─ mbr_npcs: Relationship system
  ├─ mbr_marriage: Family mechanics
  ├─ mbr_loot: Item rarity and affixes
  └─ mbr_crafting: Quality-based crafting

Layer 3: ENGINE (src/, lib/, irr/)
  ├─ Rendering: 3D graphics (Irrlicht/OpenGL)
  ├─ Networking: Multiplayer support
  ├─ Physics: Collision and movement
  ├─ Audio: Sound system
  ├─ Modding API: Lua interface
  └─ Core Systems: Built on Luanti 5.16.0

🚀 HOW TO BUILD & RUN
════════════════════════════════════════════════════════════════════════

Configure:
  $ cmake . -DRUN_IN_PLACE=TRUE -DCMAKE_BUILD_TYPE=Debug

Build:
  $ make -j$(nproc)

Run Client:
  $ ./bin/moonbrook_ridge

Run Server:
  $ ./bin/moonbrook_ridgeserver

That's it! No game selection, no mods to manage.
You're running MoonBrook Ridge - a complete game.
